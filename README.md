# RacketParser

A parser for a calculator language

## How to Use

1. Pull this repo down to your local machine.
2. Open the 'main.rkt' file.
3. Enter the file path for the file you with to parse as the parameter in the `parse` function. **Note**: The file you choose should be in the same directory.
4. The console will print 'Accept' if the source code you parsed is syntactically correct, else it will throw an error with the offending token and line number.
